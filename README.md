



 Overview

This project focuses on analyzing real-world COVID-19 data to understand trends, patterns, and relationships between confirmed cases, deaths, and recoveries.
It uses Python, Pandas, and visualization libraries to perform Exploratory Data Analysis (EDA) and generate meaningful insights.



 Objectives

* Analyze COVID-19 dataset
* Perform data cleaning and preprocessing
* Identify trends over time
* Compare country-wise impact
* Visualize data using charts and graphs



 Dataset

* Source: Public COVID-19 dataset (GitHub dataset repository)
* Data includes:

  * Date
  * Country
  * Confirmed cases
  * Deaths
  * Recovered cases



 Tools & Technologies

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly



 Project Workflow

 1. Data Collection

* Loaded dataset directly from online source

2. Data Preprocessing

* Renamed columns
* Converted date format
* Handled missing values
* Created Active cases column

 3. Exploratory Data Analysis (EDA)

* Total confirmed, deaths, recoveries
* Statistical summary

 4. Trend Analysis

* Analyzed COVID-19 cases over time

 5. Country-Based Analysis

* Identified top affected countries

 6. Data Visualization

* Line chart → Trend over time
* Bar chart → Top countries
* Pie chart → Case distribution
* Scatter plot → Relationships
* Heatmap → Correlation

 7. Interactive Visualization

* Used Plotly for dynamic charts



Results & Insights

* Observed growth trends of COVID-19 cases
* Identified highly affected countries
* Found strong correlation between confirmed cases and deaths
* Visualized pandemic patterns effectively



 How to Run the Project
